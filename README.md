## Controls
### Camera
- W: Forward
- A: Left
- S: Backwards
- D: Right
- R: Up
- F: Down
- Q/E: Rotate
- 1-4: Different Camera Positions


### General
- Spacebar: Toggle Timestop
- P: Reset


## Standard Units

Everything is measured in the diameter ping pong balls (PPB), defined as 40mm
